Time in the Zones

Convert and view time across multiple timezones

Query Parameter Usage

  • ?time=14:30 - Show specific time (uses today's date)
  • ?time=2025-12-25T14:30 - Show specific date and time
  • ?timezone=America/New_York - Change reference timezone
  • ?time=14:30&timezone=Asia/Kolkata - Combine both parameters

Click the current URL to copy it, then modify the parameters to set a specific time or timezone.
